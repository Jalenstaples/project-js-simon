### Write temporary text:

```javascript
$('#id').text('Your text here');
```

### jQuery

Look up [jquery.org](https://jquery.org).

This adds CSS to the element with ID 'green-button' in the HTML document:
```javascript
$('#green-button').css({
  color: 'purple',
  height: '100px',
  background-color: 'green',
})
```
